---
title: Glossary
---

#### Brand theme

Includes color scales, functional colors, spacing, fonts, etc.

#### Color mode

The appearance mode set by the operating system (light, dark, auto).

#### Scheme

Name for the "theme" in the UI appears to user in Appearance settings.

#### Theme-ability

The ability to apply brand, color modes, and variants.

#### Variant

A set of styles based on a specific function (success, info, danger)
